# How to use
1. Download the mod editor on the [Epic Games Store](https://store.epicgames.com/en-US/p/leap-mod-editor?lang=en-US). **The mod editor is free but takes a lot of place, 120Gb.**
2. Clone/download this repository.
3. Go in your mod editor folder, then go in `\ProjectY\Mods` and empty that Mods folder.
4. Put all the files from this repository in the Mods folder.
5. When you'll launch the game you should see all the mods in the mod editor.

## How to use in development mods
1. Look at the other branches of this repository.
2. Repeat the previous steps.

# How to contribute
0. Read the [Doc](https://docs.github.com/en/get-started/quickstart/contributing-to-projects).
1. Fork this repository.
2. Clone your fork.
3. Create a new branch.
4. Make your changes.
5. Commit your changes.
6. Push your changes.
7. Create a pull request with a description of your mod.

## Attention : 
- **Do not** steal other people's work.
- **Do not** mods that are not yours.
- Please don't make a pull request for a mod that is not working.
- Merge requests will be accepted when I'll have time to test them.
- Everyone will be able to see your changes, so don't put your personal information in the code.

## Notes :
- @me on discord if you need help or if you want me to check your pull request.
- If you don't understand a part of that readme, please ask me on discord.
- If you want me to explain one of my mods, please ask me on discord.
